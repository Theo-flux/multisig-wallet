# MultiSig Wallet

A decentralized multi-signature wallet smart contract built with Solidity and Foundry. Requires multiple approvals to execute transactions, enhancing security for shared funds or DAO treasuries.

## Quick Start

```bash
# Install Foundry
curl -L https://foundry.paradigm.xyz | bash

# Clone and setup
git clone https://github.com/Theo-flux/multisig-wallet.git
cd multisig-wallet
make install
```
